# DevLog (Developer Log)
**DevLog**는 개발자들을 위한 개인 프로젝트 관리 및 기술 블로그 플랫폼입니다. 자신의 프로젝트를 체계적으로 정리하고, 마크다운을 활용해 기술 포스트를 작성하며 성장을 기록할 수 있습니다.
## 주요 기능
- **프로젝트 관리**: 개인 프로젝트의 상세 정보를 등록하고 진행 상황을 관리합니다.
- **마크다운 블로그**: `@uiw/react-md-editor`를 사용하여 풍부한 마크다운 기반의 기술 포스트를 작성합니다.
- **사용자 인증**: JWT 기반의 회원가입 및 로그인 기능을 제공하며, 보호된 라우트(Protected Routes)를 통해 보안을 유지합니다.
- **마이페이지**: 개인 프로필 정보와 내가 작성한 글, 프로젝트를 한눈에 확인합니다.
- **반응형 디자인**: 모바일과 데스크탑 환경 모두에 최적화된 세련된 인터페이스를 제공합니다.
## 기술 스택
### Frontend
- **Framework**: React 19 (Vite 기반)
- **Routing**: React Router Dom v7
- **HTTP Client**: Axios
- **Editor**: React Markdown Editor
- **Styling**: Vanilla CSS
